# AI 行业热点新闻日报

**日期：2026年5月15日**

---

## 一、重要产品发布与功能更新

### 1. OpenAI 在 ChatGPT 全面开放 GPT-4.1 系列

**一句话摘要：** OpenAI 宣布 GPT-4.1 正式登陆 ChatGPT，Plus/Pro/Team 付费用户即日起可直接调用，同时推出 GPT-4.1 mini 作为免费层超额 fallback 模型，并上线 Safety Evaluations Hub 公开安全基准数据。

**信息来源：** CSDN AI 日报 (https://blog.csdn.net/He_r_o/article/details/147989940)

---

### 2. Google DeepMind 发布 AlphaEvolve：Gemini 驱动的算法设计代理

**一句话摘要：** DeepMind 推出基于 Gemini 2.5 系列的跨语言多模态编码代理 AlphaEvolve，可在数小时内自动生成高质量近似最优算法，在图着色、约束满足等 12 个经典难题上平均超越人类专家基线 8.3%，并开源评测框架。

**信息来源：** CSDN AI 日报 (https://blog.csdn.net/He_r_o/article/details/147989940)

---

### 3. OpenAI API 支持原生 PDF 文件输入

**一句话摘要：** OpenAI 在开发者社区宣布 API 首次支持 PDF 作为 file_id 直传格式，无需预先转为文本或多页图片，降低了文档智能化门槛，为 RAG、合同审核、科研论文导航等场景提供即插即用的后端能力。

**信息来源：** OpenAI 开发者公告 (https://blog.csdn.net/He_r_o/article/details/147989940)

---

### 4. Anthropic Claude Code 年化收入突破 25 亿美元

**一句话摘要：** Anthropic 的命令行编程工具 Claude Code 成为公司历史上增长最快的产品，年化收入已超 25 亿美元，在 AI 编程工具市场份额达 54%，全球 4% 的 GitHub 公开代码提交由其完成。

**信息来源：** 今日头条 (http://m.toutiao.com/group/7639907102697210419/)

## 今日趋势点评

**2026年5月15日**，AI 行业呈现三大趋势：一是**模型能力持续进化**，GPT-4.1、AlphaEvolve 等新品表明头部玩家在编码、推理、算法设计等垂直领域深化布局；二是**商业落地加速**，Claude Code 的爆发式增长与 Windsurf 的高价收购显示 AI 编程工具已进入规模变现阶段；三是**安全合规体系加速完善**，中国连发安全治理框架与多项国家标准，AI 治理正从"先发展后治理"转向"同步建设"，这对行业长期健康发展具有重要意义。
